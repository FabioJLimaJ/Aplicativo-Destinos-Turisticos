*Funcionalidades*

Login
Campo de usuário
Campo de senha
Botão "Entrar"
Validação com credenciais fixas no código
Mensagem de erro em caso de dados incorretos
Link para tela de cadastro
Cadastro de destino
Nome do destino
País ou região
URL de referência
Botão "Salvar" para armazenar o destino
Link para acessar a lista
Lista de destinos
Exibição em formato de cards
Cada item contém:
Nome
Região

Ações disponíveis:

"Explorar": abre o site dentro do aplicativo
"Excluir": remove o item da lista
WebView
Exibe a página do destino dentro do app
Não utiliza navegador externo
Possui botão para voltar à lista
Parte técnica
Utilização de WebView para carregar as páginas
Configuração básica do WebView
Tecnologias utilizadas
Android Studio
Java ou Kotlin
XML

*Como executar o projeto*
Clonar o repositório
Abrir no Android Studio
Executar em um emulador ou dispositivo físico
